# I18n Agent Instructions

When editing translations in `i18n/locales/` or `packages/underverse/locales/`, read the locale-specific rule file before changing any locale JSON:

- `vi`: read `i18n/Vietnamese_rule.md`
- `ko`: read `i18n/Korean_rule.md`
- `ja`: read `i18n/Japanese_rule.md`

For `en`, use concise, natural product UI English and keep terminology aligned with the module.

Always translate by UI context, not literal word matching. Preserve JSON keys, placeholders, ICU syntax, and namespace structure exactly.
