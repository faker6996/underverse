Place font files here if you need non-Latin glyphs (e.g., Vietnamese + CJK).

Recommended:
- Noto Sans (Latin + Vietnamese): NotoSans-Regular.ttf, NotoSans-Bold.ttf
- Noto Sans CJK (Chinese/Japanese/Korean): NotoSansCJKsc-Regular.otf (or equivalent)

FFmpeg/libass will load fonts from this folder when burning subtitles, avoiding system font errors.
